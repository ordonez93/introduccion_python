iniciamos por primera vez
- git init => inicia un proyecto en la carpeta actual
- git status => muestra el estado actual del repositorio 
    comparado con la versio anterior
- git add . => agrega todos los cambios
- git commit -m 'iniciamos repositorio'
- git branch -M main
- git remote add origin https://github.com/ordonez93/introduccion_python.git
- git push -u origin main 
cuando el proyecto ya esta creado
- git add . 
- git commit - m 'mensaje para el commit'
- git push 

como clonar un repositorio (desde ventana de power shell)
- git clone https://github.com/ordonez93/introduccion_python.git
